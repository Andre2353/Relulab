# Biolab - Site Institucional

Projeto de uma landing page para a empresa Biolab, focado em biotecnologia e saúde.

## 🚀 Tecnologias
* HTML5
* CSS3
* Google Fonts (Roboto)

## 📁 Organização
* `index.html`: Estrutura do site.
* `/styles`: Arquivos de design.
* `/assets`: Imagens e logos.

## ⚙️ Como abrir
Basta abrir o arquivo **index.html** no seu navegador.
